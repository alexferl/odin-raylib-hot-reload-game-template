# odin-raylib-hot-reload-game-template

This is a [Odin](https://odin-lang.org) + [raylib](https://www.raylib.com/) 3D game template with hot reloading.

## Usage
Setup the development environment first:
```shell
make
```

Build and start the game:
```shell
make dev
```
